# pyopath

`pyopath` is a  WIP **full-compatibility clone of Python's `pathlib`** implemented in **Rust** via **PyO3**.

End goal is to be a drop-in replacement for `pathlib` with better performance.

## Installation

```bash
uv add pyopath
```
